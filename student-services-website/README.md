# 🎓 EduReach University — Student Services Website

A professional, fully responsive Student Services website built with **pure HTML, CSS, and JavaScript** (no frameworks).

## 📁 Project Structure

```
student-services-website/
├── index.html          ← Homepage
├── pages/
│   ├── services.html   ← All student services (searchable)
│   ├── about.html      ← Mission, vision, team
│   └── contact.html    ← Contact form with validation
├── css/
│   └── style.css       ← All styles + dark mode + responsive
├── js/
│   └── script.js       ← Mobile menu, dark mode, validation, animations
├── assets/             ← Add images/icons here
├── .vscode/
│   └── settings.json   ← VS Code recommended settings
└── README.md           ← This file
```

## 🚀 How to Run Locally

### Option 1 — VS Code Live Server (Recommended)
1. Install the **Live Server** extension in VS Code
2. Right-click `index.html` → **"Open with Live Server"**
3. Your browser opens automatically at `http://127.0.0.1:5500`

### Option 2 — Open directly in browser
- Double-click `index.html` to open in your browser

## ✨ Features
- ✅ Responsive design (mobile, tablet, desktop)
- ✅ Dark / Light mode toggle (saves preference)
- ✅ Mobile hamburger menu
- ✅ Live search filter on services page
- ✅ Contact form with JS validation
- ✅ Scroll-triggered fade-in animations
- ✅ Animated stat counters

## 🌐 Deploy to GitHub Pages
1. Push all files to a GitHub repository
2. Go to **Settings → Pages → Branch: main → Save**
3. Live at: `https://yourusername.github.io/repository-name/`
